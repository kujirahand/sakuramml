名称 書式 説明 キー
Array Array name[=(value,value,value...)]; 配列変数(name)を作成し、(value,value,value...)で初期化する スクリプト

【解説】

配列変数(name)を作成し、(value,value,value...)で初期化する。
配列変数同士の足し算は、"A = (A, B); "のように、( )の中に配列変数を記述することによる行う。
配列の１要素を取り出すには、"A(要素番号)"のように参照する。

【サンプル】

//Array のサンプル
Array A = (0,1,2,3)
Array B = ();//空の配列
B = (A,4,5,6);//配列の足し算
Print(B);//要素の内容を表示=(0,1,2,3,4,5,6)
Int C = B(3);//要素１つを取り出す
//ここまで
