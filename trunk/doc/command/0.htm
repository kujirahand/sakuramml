l l(n) 音の長さをn分音符で指定(初期値4) 音符属性

【解説】

音符の基本となる音の長さを、ｎ分音符で指定する。数値と音の対応表を以下に示す。
音符 表記方法
全音符 l1
四分音符 l4
八分音符 l8
十六分音符 l16
付点四分音符 l4.
四分音符の３連符 l12
八分音符の３連符 l24

音の長さの計算は、
（基本音符）×（分割したい個数）＝（求めたい音符の長さ）
で求められる。

【サンプル】

//l のサンプル
l4 cdef l8 cdef l16 cdef l32 cdef
//ここまで
