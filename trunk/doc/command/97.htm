NoteNo NoteNo(mml) mml に書かれたノートのノート番号を返す 変数

【解説】

NoteNo(mml)で、mmlに書かれた、ＭＭＬのノート番号を取得します。

このNoteNoで得られる値には、カレントトラックの、オクターブ指定及び、"System.KeyFlag"、"TimeKeyFlag"の、影響を受けます。
（但し、"System.KeyShift(=Key)"、"TimeKey"の、キーシフトの値は、適用されません。）

【サンプル】

//NoteNo のサンプル
Int N = NoteNo(o5e)
Print(N);
//ここまで
