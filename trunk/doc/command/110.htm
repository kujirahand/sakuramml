UseKeyShift UseKeyShift(on|off) KeyやTimeKeyやTrackKeyの適用をオンオフする。リズム演奏のときなどに利用。 システム

【解説】

KeyやTimeKeyやTrackKeyの適用をオンオフする。これをオフにしても、KeyFlagの値は適用されるので注意。
リズム楽器の演奏の時は、KeyShiftすると、楽器番号が変わってしまうので、これを、offにして使う。

【サンプル】

//UseKeyShift のサンプル
Key(2) c
UseKeyShift(off)
c
UseKeyShift(on)
c
//ここまで

【関連項目】
