名称 書式 説明 キー
TempoChange TempoChange(t1[[,t2],len]) テンポをt1からt2へlen(ステップ数)の長さで変更する。t2を省略すると、現在のテンポからt1までの推移。lenを省略すると全音符の長さになる。 メタイベント

【解説】

テンポをt1からt2へlen(ステップ数)の長さで変更する。t2を省略すると、現在のテンポからt1までの推移。lenを省略すると全音符の長さになる。

【サンプル】

//TempoChange のサンプル
TempoChange(120, 80, !1);l4 cdef gfed //120から80へ、テンポを滑らかに変更する
TempoChange(160,!2);l4 cdef gfed //現在のテンポから、160へ２分音符の間に変更する
TempoChange=160; l4 cdef gfed // 全音符の間に、テンポを160に滑らかに変更する
//ここまで
