名称 書式 説明 キー
KeyFlag KeyFlag[+|-](note)|KeyFlag=(a,b,c,d,e,f,g) 臨時記号を設定する システム

【解説】

臨時記号を設定する。
設定の方法は、直接ノートを指定する方法と、全てのノートについて指定する方法と、２種類ある。

・直接ノートを指定する方法

KeyFlag+(ドファ) // ドファに＃記号をつける
KeyFlag-(ミシ) // ミシに♭記号をつける

・全てのノートについて指定する方法
KeyFlag=(-2,0,0,-2,0,0,0) // 沖縄音階を再現

【サンプル】

//KeyFlag のサンプル

KeyFlag+(ドファ)
ドレミファソラシ

KeyFlag=(-2,0,0,-2,0,0,0);
ドレミファソラシ＞ド
