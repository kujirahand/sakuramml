SoundType SoundType=v ユーザーが使用している音源のタイプを得る。v=0:GM/1:GS/2:XG/3:i-mode システム

【解説】

ユーザーがエディタで設定した音源の種類を得る。
音源ごとに違う処理を実装したい場合などに有効。

【サンプル】

//SoundType のサンプル

Int TYPE_GM = 0;
Int TYPE_GS = 1;
Int TYPE_XG = 2;

If(SoundType = TYPE_GM){ @1 cdef }
If(SoundType = TYPE_GS){ @1,8 gfed }
If(SoundType = TYPE_XG){ @1,0,16 cgcg }

//ここまで
