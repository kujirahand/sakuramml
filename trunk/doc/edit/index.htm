エディタの使い方

　サクラには、ボタン一発で、演奏を開始したり、命令などを挿入したりできる便利な、テキストエディタがついています。このテキストエディタは、テキストで音楽を表現するのを支援する数々の機能がついています。


簡単な使い方

  1. テキストエディタ(上図１)に、『ドレミ』などの演奏情報を書きます。
  2. 再生ボタン(上図２)を押します。
  3. もし、演奏情報に間違いがあれば、メッセージが(上図３)に表示されます。
  4. 演奏モニター（上図４）が表示され、演奏が始まります。

(4)演奏モニターについて

　カラフルな演奏モニターです。左側にチャンネルごとのパラメーター。右側に鍵盤は押されている鍵盤に色がつきます。色のカスタマイズをすることができます。

(5)鍵盤入力、(6)五線譜入力について

　鍵盤や五線譜をマウスでクリックすると、エディタに、ドレミの演奏情報が挿入されます。五線譜入力の場合、シャープを入力するには、[SHIFT]キーを押します。

エディタに関する、その他の機能

  • (3)メッセージタブ・・・メッセージや、ヘルプなどを表示します。
  • (5)鍵盤入力・・・画像で示されたキーボードをクリックすることにより、ドレミのテキストを挿入できます。
  • (6)楽譜入力・・・五線譜上をクリックすることにより、ドレミのテキストを挿入できます。
  • 便利タブ・・・音色を視聴したり、命令を挿入できます。
  • しおり・ブックマーク・・・エディタの任意の部分に、しおりを設定できます。
  • 曲掲示板の巡回機能・・・ユーザーが作った曲をすばやくダウンロードして聞けます。
  • MIDI素材掲示板・・・MIDI素材掲示板へMIDIデータを簡単に登録できます。
  • クイックヘルプ・・・エディタ上で単語をダブルクリックすると、ヘルプが表示されます。
  • ＤＤＥで制御・・・プラグインなどから、サクラエディタを操作することができます。
