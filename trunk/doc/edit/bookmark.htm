しおり・ブックマーク

　サクラのエディタでは、編集中の行数を記憶しておく簡易ブックマーク機能と、編集中の任意の部分に名前をつけておく、しおり機能があります。

簡易ブックマーク

　簡易ブックマークを使うと、任意の個所（最大１０箇所）に、ブックマークを設定し、瞬時にその場所へ戻ることが出来ます。

　簡易ブックマークの設定の方法は、エディタ上で、右クリックすると、「ブックマークを登録」というメニューが出てきますので、そこで、登録番号を選択すると、ブックマークを設定できます。

　また、ショートカット（Borland Delphiと同じ）にも対応しており、Shift + Ctrl + 数字、で、ブックマーク登録、Ctrl + 数字、で、ブックマークへ戻ることが出来ます。

しおり機能

　しおり機能とは、コマンド挿入タブにあり、エディタ任意の部分に、名前をつけておき、後から、その部分へすぐにジャンプできるようにする機能です。

　しおり機能で、しおりを作るには、エディタの、任意の個所の、行頭に、『//_名前』のようなコメントを書いておくか、エディタ上で、右クリックすると出てくる、『しおりの登録』を使っても行うことが出来ます。
