MasterVolume

■概要

　この機能は、各MIDIデバイスの音量を設定します。


■書式

MasterVolume(INT iVolume);
MasterVolume_EX(INT idDevice, INT iVolume);
[備考]
※この命令は、以下のMMLと同等です。
SysEx = $F0, MakerID_RealTime,idDevice, RT_DeviceCont, RT_DeviceCont_MasterVolume, iVolume(2Byte), $F7

■引数

引数名 初期値 範囲 機能
INT iVolume 16383 0〜16383 MIDIデバイスの音量を指定します。
値による音量カーブは、値の二乗に比例します。
INT idDevice $7F $00 〜 $7F MIDIデバイスのDeviceIDを指定します。
MIDI規格では、Bloadcastである$7Fを推奨しています。


■返り値

無し


■説明

　MIDIデバイスの音量をiVolumeに設定します。
　MasterVolume()関数は、idDeviceを$7F(Bloadcast)にしてMasterVolume_Ex()関数をコールしている為、全てのMIDIデバイスがメッセージを受信します。

■使用例

List.1 

MasterVolume(100);
TR(1)	o4 l4 cdefgabc
