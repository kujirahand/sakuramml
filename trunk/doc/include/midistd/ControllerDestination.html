Controller Destination Setting

概要

Control Change Message, Channel Pressure及びPolyphonic Key Pressureによる機能の設定を可能にします。

詳細

コントロールチェンジ、アフタータッチ等のコントローラ・ソースに、１つ以上の機能を割当てます。 このメッセージは、セットに機能と可変範囲の両方を含み、MIDIデバイスの応答の大きさを定義します。 例えば、ピッチかもしれないし、ＬＦＯかもしれないし、カットオフ周波数かもしれません。
このController Destination Settingメッセージは、 単一のメッセージで複数のセット（機能と可変範囲）を含むことが出来るので、メッセージの長さは可変です。

各チャンネルにおいては、最後に受信したメッセージのみ有効となります。 すなわち、以前に別の機能が設定されている場合でも、 最後に受信したコントローラー・ディスティネーション・セッティングのみが有効になります。

又、GM System Level.2(GM2)では、リズム・トラックでは受信しない事を推奨しています。

■Channel Pressure

各チャンネルに対して、チャンネル・プレッシャーの効果を割り当てます。
  • RT_CD_ChPressure

■Polyphonic Key Pressure

各チャンネルに対して、ポリフォニック・キー・プレッシャーの効果を割り当てます。
GM System Level.2(GM2)では、この機能は要求されていません。
  • RT_CD_PolyPressure

■Control Change message

各チャンネルに対し、コントローラー１種の効果を割り当てます。
各チャンネルにおいて、同時に複数のコントローラーを設定することはできません。 他のコントローラーのコントローラー・ディスティネーション・セッティングが 既に設定されている場合には、それはリセットされます。 ただしチャンネル・プレッシャーのコントローラー・ディスティネーション・セッティングの設定は変化しません。
  • RT_CD_CC
