MIDI Standard

"midistd.h"及び、"midistd.h"内にて呼び出される各種定義ファイルは、 MIDI規格にて標準化されており、且つ、Sakuraの標準命令セットに無い機能について関数化し、 ユーザーに提供されるMIDIの機能を補完する事を目的としています。
更に、各種定義ファイルには、ユーザーの関数製作を助ける為、各種機能やIDの番号等を定数化しています。
（定義内容の詳細については本マニュアルには記載していませんので、各種関数の定義ファイルを参照してください。）

又、本マニュアルにて、Sakura及びMIDIによるDTM活動に必要な情報を記載しています。

■本マニュアルの表中の"GM1","GM2"について。
○ … General MIDIの仕様書にて実装が義務つけられている機能。
△ … General MIDIの仕様書にて実装が推奨されている機能。
× … General MIDIの仕様書にて実装は任意とされている機能。

Channel Voice Message

MIDIデバイスの1〜16の各チャンネルを制御する為のメッセージす。
大別すると、以下の５つつのメッセージがあります。
  • Note on / Note off Message
  • After Touch Message
  • Control Change Message
  • Program Change Message
  • Pitch Bend Change Message


Channel Mode Message

MIDIデバイスの1〜16の各チャンネルのモードを設定する為のメッセージす。
General MIDI(GM)規格に対応した音源デバイスは、初期モードは３となります。
  • Mode1
  • Mode2
  • Mode3
  • Mode4


System Exclusive Message

チャンネルに関係なく、MIDIデバイスのシステム全体に共通するメッセージです。
各楽器メーカーにID番号(1 or 3Byte)が割り振られており、MIDIデバイス固有の機能を設定する事が可能です。
その為、ID番号によって様々なフォーマットがあります。
又、$7D〜$7Fの３つのメーカーID番号は、全てのMIDIデバイスに共通するメッセージとして割り振られいます。
$7Dは、教育機関、研究機関等のテスト用で商用のMIDIデバイスでの使用は禁止されています。
$7E〜$7FはUniversal Exclusive Messageで、MIDI規格の拡張として使用されます。

命令一覧 (Sakura標準命令)

  • SysEx = $F0, Data1, Data2, …, DataN, $F7
  • SysEx$ = F0, Data1, Data2, …, DataN, F7
ID番号Maker
$7DTest
$7EUniversal Non Real Time Exclusive Message
$7FUniversal Real Time Exclusive Message
