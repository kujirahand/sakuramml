インストール／アンインストールの方法

　テキスト音楽「サクラ」は、拡張子".MML"への関連付け以外には、レジストリを利用しませんので、インストールが簡単で、アンインストールした時、レジストリを汚すことがありません。

【インストール】

【インストーラーを使う場合】
setup.exe をダブルクリックしてください。
【インストーラーを使わない場合】
特に作業は必要ありません。
【初めて起動する場合】
　サクラを初めて起動すると、拡張子".MML"へ、サクラを関連付けるかどうか尋ねます。もし、ここで関連付けを「いいえ」にした場合は、エディタの、メインメニューにある[設定] - [関連付け]のメニューから、関連付けしたり、解除したりを選択できます。
　それから、サクラは、パソコンのMIDI音源を利用して、音楽を鳴らすので、MIDI音源の設定が必要になります。MIDI音源の設定については、分からない場合は、こちら（音が鳴らない）をご覧下さい。

【アンインストール】

【インストーラーを使ってインストールした場合】
コントロールパネルの、プログラムの追加と削除よりアンインストールしてください。
【インストーラーを使わずにインストールした場合】
サクラは、関連付け以外には、レジストリを利用していませんので、アンインストールは、以下の手順にしたがって行えば、クリーンにアンインストールできます。
１．メインメニューにある、[設定]より、[関連付け]-[関連付けを解除する]で、レジストリの関連付け情報を削除します。
２．サクラの実行ファイルやマニュアルのあるフォルダをゴミ箱に捨てます。
参考までに）ＭＭＬの関連付けに使っているレジストリキー
HKEY_CLASSES_ROOT\.mml
HKEY_CLASSES_ROOT\mmlFile

【エディタの設定情報について】

　サクラのエディタの設定情報は、サクラの実行ファイル"sakura.exe"と同じフォルダへ、"sakura.ini"として設定ファイルが作成されます。サクラのエディタの設定を初期状態に戻したい場合は、このファイルを削除すれば、初期状態に戻すことが出来ます。

【スキン情報＆カラーリングについて】

　スキン＆カラー情報は、skin フォルダに保存されています。
