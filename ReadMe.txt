　サクラは、『ドレミ』のテキストから、音楽を作るソフトです。
　クラシックからポップス、果ては、実験音楽まで、あらゆるジャンルの音楽を、ワープロ感覚で作ることが出来ます。
──────────────────────────────
　詳しい使い方は、docフォルダの、index.htmをご覧下さい。

【インストール】
──────────────────────────────

ファイルを解凍したら setup.exe をダブルクリックしてください。

【初めて起動する場合】

　サクラを初めて起動すると、拡張子".MML"へ、サクラを関連付けるかどうか尋ねます。もし、ここで関連付けを「いいえ」にした場合は、エディタの、メインメニューにある[設定] - [関連付け]のメニューから、関連付けしたり、解除したりを選択できます。
　サクラは、パソコンのMIDI音源を利用して、音楽を鳴らすので、MIDI音源の設定が必要になります。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【アンインストール】
──────────────────────────────
【インストーラーを使ってインストールした場合】

コントロールパネルの、プログラムの追加と削除よりアンインストールしてください。

【インストーラーを使わずにインストールした場合】

サクラは、関連付け以外には、レジストリを利用していませんので、アンインストールは、以下の手順にしたがって行えば、クリーンにアンインストールできます。

１．メインメニューにある、[設定]より、[関連付け]-[関連付けを解除する]で、レジストリの関連付け情報を削除します。
２．サクラの実行ファイルやマニュアルのあるフォルダをゴミ箱に捨てます。
